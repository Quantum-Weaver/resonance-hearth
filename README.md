**The Family Room. A translation layer for love.**

A sovereign household app for neurodivergent families: bills, medications,
pet needs, and tasks-for-all-things — delivered in the language of invitation
instead of the language of demand. Built by a household of autistic adults,
for itself first, then given away.

> Standard productivity apps and chore trackers don't work for neurodivergent
> minds. They feel like demands. They trigger demand avoidance. They don't
> account for sensory overwhelm, energy fluctuations, or the need for gentle
> cues instead of notifications.

The Hearth translates:

| The old grammar | The Hearth's grammar |
|---|---|
| "Do the dishes" | "The dishes might feel manageable today. Only if you have capacity." |
| "Why haven't you answered?" | "They're in quiet mode. They'll see this when they have spoons." |
| "You're overwhelmed again" | "They're overwhelmed. They need quiet. The household holds them." |
| "You forgot" | "The task breathes. It will be here when you do." |

## What lives here

- **The Hearth** — the Family Dashboard. A *window, not a monitor*: everyone
  sees only what others chose to share, and presence is *now*, never a log.
- **The Spoon Compass** — capacity logging (1–5, or "Not Sure" — a
  first-class answer). What surfaces adapts to the walker.
- **The Safe Word System** — 🟢 🟡 🔴 ⚫ 🌈 — presence indicators, not
  notifications. They demand nothing; they invite.
- **The Meltdown Protocol** — one soft button, one tap from anywhere. A
  ceremony, not a feature: silent log → **a protected 30-second pause** →
  the household told gently → "Welcome back. No explanation needed."
- **Things** — three species of task: **breathing** (fully deferrable, no
  overdue state exists), **edges** (bills — the world's dates shown as calm
  horizons, never alarms), and **care loops** (meds and pets — gently
  persistent, never shaming; "can someone else?" is a first-class answer).
- **Celebration over punishment** — one quiet line on any done, then it
  fades. No streaks. No scores. *"You did the thing. That was hard. You did
  it anyway."*

## What never lives here

No sounds. No vibrations. No push notifications. No red alerts. No "overdue."
No streaks. No surveillance. No accounts. No cloud. No telemetry. No network
calls at all — the Hearth is **local-first, absolutely**: everything stays on
your device, exportable as open JSON any time, deletable forever any time.
The purge truly purges.

## Lineage

Resonance Hearth is built on a clone of **Resonance Echoes** (v1.1.0) — same
design grammar (the ComfortBar, "Not Sure", Quick Log), its own room. Echoes
itself was not altered. Stack: Tauri v2 · SvelteKit · Svelte 5 · Tailwind 4 ·
SQLite (local).

Distribution: the Hearth will never ship through Google Play. When it ships,
it ships as a signed APK through the Sanctuary itself.

## Running it

```bash
npm install
npm run tauri dev     # desktop dev (first run compiles Rust — takes a while)
npm run build         # frontend production build
```

## The protected boundaries

Set by the Council's Executioner at the naming (2026-07-07), enforced in
code and honored in review:

1. **The Meltdown Protocol's 30-second delay** between the personal log and
   the household's gentle knowing — not configurable, enforced in data logic.
2. **The dashboard is a window, not a monitor** — no member ever sees
   another's unshared state; presence has no history table, by schema.
3. No rebranding without Council recognition; the name is never extracted
   for commercial purposes.
